Question:

What distinguishes normal HTML elements from semantic HTML elements? Explain the disparity between closing tags and self-closing tags in HTML. What distinguishes inline elements from block elements in HTML?

Answers;

  • Normal vs. Semantic HTML Elements:Normal HTML elements (non-semantic) like <div> and <span> do not convey any meaning about their content to the browser or the developer.Semantic HTML elements like <form>, <table>, and <article> clearly define their content and purpose to both the browser and the developer, making the web page more accessible and the code more readable.
  • Closing Tags vs. Self-Closing Tags:Closing tags are used for elements that wrap content, like <p>...</p> or <div>...</div>. They require an opening tag and a corresponding closing tag with a forward slash before the tag name. Self-closing tags, also known as void elements, like <img />, <br />, and <input>, do not wrap content and therefore do not need a closing tag. They are used for elements that stand alone.
  • Inline vs. Block Elements:Block elements like <div>, <p>, and <h1> start on a new line, take up the full width available, and can contain other block or inline elements. Inline elements like <span>, <a>, and <img> do not start on a new line and only take up as much width as necessary. They are typically used within block elements to format text or insert content.

Visual representation using HTML code;

HTML:
                
                    <div>
                       <p>This is a paragraph, a block element that takes the full width.</p>
                    </div>
                
            
                
                    <p>This is a span element, an inline element within a paragraph.</p>
                    
                
            
                
                    <br />
                       <img src="image.jpg" alt="Description" />
                    The <img /> and <br /> tag are self closing tags.

In the example stated above, <div> and <p> are block elements. <span> is an inline element. <br /> and <img /> are self-closing tags. The use of semantic elements and proper tag types helps create a structured and accessible web page.
